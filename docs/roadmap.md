## W2 预规划（AI 教练接入）

**目标**：深蹲完成后，AI 教练根据本组错误数据生成个性化中文指导语。

**关键决策**：
- 本地规则引擎处理实时反馈（< 20ms），Claude API 仅在组间休息时调用（异步）
- Prompt 输入：`{ exercise: "squat", reps: 5, errors: ["notDeepEnough×3", "backTooForward×1"] }`
- 输出：2-3 句中文指导，口语化，避免专业术语

**风险**：
- Claude API 延迟 300-800ms，需异步加载避免阻塞 UI
- Prompt 设计需要在真实用户场景下迭代，W2 结束时应有 10+ 真实对话样本
